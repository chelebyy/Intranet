# Product Guidelines: Kurumsal İntranet Web Portalı

## 1. Communication Style & Tone

*   **Tone:** Professional and Formal. The portal should communicate reliability and authority.
*   **Prose Style:**
    *   Use clear, direct, and grammatically correct Turkish.
    *   Avoid slang, jargon (unless technical/domain-specific), and overly casual language.
    *   Instructions should be polite but imperative (e.g., "Kaydet" instead of "Lütfen kaydedin").
*   **Language Policy:**
    *   **UI Strings:** Turkish (e.g., "Kullanıcı Yönetimi").
    *   **Code/System Comments:** English.
    *   **Error Messages:** Informative in Turkish for users; technical details in English for logs.

## 2. Visual Identity & Design Principles

*   **Design System:** Built on a foundation of **Shadcn/UI** and **TailwindCSS**.
*   **Principles:**
    *   **Consistency:** Use established component patterns for all features.
    *   **Clean & Minimalist:** Prioritize essential information and reduce visual clutter.
    *   **Corporate Palette:** Primary focus on blues, grays, and whites to maintain a professional aesthetic.
    *   **Responsive Design:** Optimized for desktop (primary) and mobile (secondary) access.
    *   **Theme Support:** Native Light and Dark mode support with high contrast.
*   **Typography:** Use modern sans-serif fonts (e.g., Inter, Gehlow) for readability.

## 3. User Experience (UX) Standards

*   **Navigation:** Intuitive sidebar navigation with clear active states.
*   **Feedback:** Provide immediate visual feedback for all user actions (e.g., loading states, success/error toasts).
*   **Forms:** Consistent layout, clear labels, and inline validation.
*   **Accessibility:** Adhere to WCAG 2.1 Level AA standards where feasible.

## 4. Brand Messaging

*   **Vision:** "Güvenli ve Modern Kurumsal İletişim" (Secure and Modern Corporate Communication).
*   **Core Values:** Security, Reliability, Efficiency, and Collaboration.
